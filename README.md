# community

This repository is the central place to report issues, request
  features, and share feedback for all projects published on
  [solorpgstudio.com](https://solorpgstudio.com/tools).

  ## What belongs here

  - **Bug reports** — something is broken or not working as
  expected
  - **Feature requests** — an idea or improvement you'd like to
  see
  - **General feedback** — thoughts on existing tools, articles,
  or the site itself

  ## Projects covered
- [5e to Nimble Monster Converter](https://www.solorpgstudio.com/tools/5e-to-nimble-monster-converter)
- [obsdian Game Library Vault](https://www.solorpgstudio.com/tools/games-library)

  ## Opening an issue

  1. [Check existing issues](../../issues) to avoid duplicates
  2. Click **New Issue**
  3. Include as much context as possible:
     - Which project or page is affected
     - What you expected vs. what happened
     - Steps to reproduce (for bugs)
     - Screenshots if relevant

  ## Notes

  - This repo contains no source code — it exists solely for
  community tracking
  - Feature requests are welcome but not guaranteed to be
  implemented
  - For direct contact, reach out at
  [hello@solorpgstudio.com](mailto:hello@solorpgstudio.com)
